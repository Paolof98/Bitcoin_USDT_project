# Analysis of the relationship between Bitcoin and Tether

The report I wrote on the relationship between Bitcoin and Tether can be accessed by this [link](https://github.com/Paolof98/Bitcoin_USDT_project/blob/main/REPORT%3A%20Analysis%20of%20the%20Relationship%20between%20Bitcoin%20and%20Tether.pdf), and it can be downloaded by clicking on the download button on the top right.

The code in this repo consists of the following files: 

1) [SQL code](https://github.com/Paolof98/Bitcoin_USDT_project/blob/main/SQLcode.sql): Data cleaning and creating tables
2) [Analysis code.R](https://github.com/Paolof98/Bitcoin_USDT_project/blob/main/Analysis%20code.R): 1st part: Statistical models to analyse generic relationship between Bitcoin and Tether
3) [Transaction types.R](https://github.com/Paolof98/Bitcoin_USDT_project/blob/main/Transaction%20types.R): 2nd part: Relationship between Bitcoin and different types of Tether transactions
4) [Addresses.R](https://github.com/Paolof98/Bitcoin_USDT_project/blob/main/Addresses.R): 3rd part: Relationship between Bitcoin and interesting addresses
5) [Days.R](https://github.com/Paolof98/Bitcoin_USDT_project/blob/main/Days.R): 4th part: Bitcoin movements during interesting days
